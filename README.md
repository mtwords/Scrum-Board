# Scrum Board
A simple scrum board written with node.js and backbone.js.

A running sample can be found on http://warm-scrubland-7411.herokuapp.com/

## Setup
Install dependencies with

```
npm install
```

## Technologies
- node.js
- backbone.js
- bootstrap

## Features
- known "todo", "in progress" and "done" categories
- add / edit new tasks
